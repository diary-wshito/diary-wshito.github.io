HugoでAsciidocフォーマット原稿を書く際の留意点

August 25, 2017 on #cms #hugo #asciidoc

Asciidoctorで変換することを前提に，Hugoでasciidoc原稿を書く際に気をつけるべき点を書き留めておく．

Level 0のタイトル

Hugoは = で始まるLevel 0のタイトルを無視するので，独立した文書として変換するときのために = で始まる行にタイトルを記述して問題ない．ただしHugo（もしくはtheme）がページタイトルとして抽出するのは，ページ冒頭に付加するFront Matterで設定する title である．

コマンドの前後の空白

Asciidocは，DocBookに比べマークアップの手間が省ける分，地のテキストとコマンドを分離するマークアップ記号がない．そのため，コマンドの前後に空白を置かなければならないという制約がある．

インライン・コードのケース

インライン・コードを書く際は，下のようにコードの前後に空白を入れる必要がある．

コード `print();` の前後に空白を入れる

ただしAsciidoctorバージョン1.5.6.1現在，句読点の直後には空白を入れなくてもコードとして認識される．

このように，`print();` 後ろだけ空白を入れれば良い．`print()` ここも大丈夫．

これを変換すると以下のようになる．
“このように，print(); 後ろだけ空白を入れれば良い．print() ここも大丈夫．”

URLのケース

残念ながらURLは句読点の後ろでも半角スペースを入れなければ，URLとして認識されない．URL以外のコマンドも同様である．

こんな時， http://gohugo.io[Hugo] は前後に空白を入れる

PrismJSを使ったコードハイライト

PrismJS は，<pre><code class="language-lisp"> で始まるコードブロックを処理する．Asciidocの記法でコードを書けば，この形式に変換される．

行番号のつけ方

行番号をつけるには，言語の後ろに半角スペースを入れて line-numbers を指定する．これにより，<code class="language-c line-numbers"> というマークアップが生成される．

```lisp line-numbers
(cffi:define-foreign-library libuv
  (:darwin  (:or "libuv.1.dylib" "libuv.dylib"))
  (:unix    (:or "libuv.so.1" "libuv.so"))
  (:windows (:or "libuv.dll" "libuv-1.dll"))
  (t        (:default "libuv")))

(unless (cffi:foreign-library-loaded-p 'libuv)
  (cffi:use-foreign-library libuv))
```

以下のように変換される．

(cffi:define-foreign-library libuv
  (:darwin  (:or "libuv.1.dylib" "libuv.dylib"))
  (:unix    (:or "libuv.so.1" "libuv.so"))
  (:windows (:or "libuv.dll" "libuv-1.dll"))
  (t        (:default "libuv")))

(unless (cffi:foreign-library-loaded-p 'libuv)
  (cffi:use-foreign-library libuv))

次のように記述しても良い．

コールアウトを実現するには

Asciidoctorはコールアウトを <b class="conum">(1)</b> というマークアップに変換する．PrismJSは，ソースコード中のマークアップをプログラミング言語の一部とみなして別の形式に変換してしまうので，ソースコード中のマークアップをそのまま残す “Keep Markup” プラグインを一緒にインストールする．あとは，スタイルシートで見栄えを設定する．

下のコードを変換してみる．

[source, c line-numbers]
.Idle basic
----
#include 
int main() {
    printf("Hello World\n");    // <1>
    return 0;
}
----
<1> コールアウト
Idle basic
#include <stdio.h>
int main() {
    printf("Hello World\n");   (1)
    return 0;
}

  1. コールアウト
